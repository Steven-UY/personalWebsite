# Hello World!!

Welcome to my personal website! This is mostly for a "professional" use case for tech jobs but
I might as well have some fun with it!

---

## Why the nim template?

This website is based off of the nim template which uses the motion-primitives component library so no I'm not
THAT talented at frontend design but maybe I'd like to think I'm solid? I stumbled across another person's website portfolio
and I really dug the aesthetic. Low and behold he used the nim template so I thought to myself why not! I think a template is a good
idea anyways, others have done it to and it's there to be used! I like that with the component library I can also customize the website
so I'm trying to still make it uniquely mine at least design wise.
